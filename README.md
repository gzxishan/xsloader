# xsloader
xsloader is a JavaScript file and module loader like requirejs but not requirejs[]()
## Documentation:[Wiki](//github.com/gzxishan/xsloader/wiki)

# 公司(Company)
[贵州溪山科技有限公司](http://www.xishankeji.com)

# 安装
### 1.初始化
```
npm install
```
### 2.构建
```
npm run build:all
```
